# Enterprise AI Agent Platform

> Java 工程师转型 AI Agent 的作品集仓库

## 项目定位
统一平台仓库，包含四个业务模块：
- **knowledge-rag**：企业知识库 RAG 问答系统
- **ticket-agent**：AI 工单助手
- **cmdb-governance**：CMDB/CI 数据治理插件
- **devops-agent**：轻量 DevOps 排障 Agent

## 技术栈

| 层 | 选择 |
|---|---|
| 业务入口 | Java 17 + Spring Boot 3.x |
| AI 服务 | Python 3.11 + FastAPI |
| 数据 | PostgreSQL 16 + pgvector |
| 协议 | REST / SSE / Tool Calling / MCP |
| 工程 | Docker Compose / pytest / JUnit |
